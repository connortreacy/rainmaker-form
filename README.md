App base
============

##How to install

1. Run bundle
2. Run middleman
3. Go to localhost:4567

##What's in it

- Haml
- Sass
- CoffeeScript
- Light Ruby back-end
- Jquery
- Susy
- Breakpoint
- Font Awesome

It's got live reload and everything!
Based on Kai's thingy: https://github.com/kaichanvong/app_base
